
# 正则表达式

## 元字符
``` shell

# 表示位置。单词的开始和结尾, \bhello\b
\b

# 表示位置。匹配字符串的开始
^

# 表示位置。匹配字符串的结束
$

# 表示字符。匹配一位数字
\d

# 表示字符。匹配一个任意的空白符，包括空格、制表符、换行符、中文全角空格等。
\s

# 表示字符。匹配一个字母或数字或下划线或汉字等。
\w

# 表示字符。匹配除了换行符以外的任意字符
.

# 表示数量。指定*前边的内容可以重复任意多次
*

# 表示数量。匹配一次到多次，大于等于一
+

# 表示数量。前面紧临的字符必须连续重复n次，不能多，也不能少。\d{8}
{n}

# 表示数量。重复n次或更多
{n,}

# 表示数量。匹配前面的字符m到n次，不能小于m，不能大于n。 m <= x <= n
{m,n}

# 表示数量。重复0次到1次
?

```

## 字符转义
如果需要查找元字符本身，使用转义字符。
