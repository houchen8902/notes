
# 其它

## QA
- **Q:** `char`类型变量打印输出是数字还是字符？**A:** 输出是字符
- **Q:** 浮点数强转为整数时，是向上转整还是向下取整？**A:** 向下取整，扔掉小数部分

## 条件真假
- **true:** 非零整数(包括非`\0`字符)、`true`、所有字符串
- **false:** 0或`\0`，`false`
- **error:** 小数、`NULL`

## 库函数

`setw()`控制后面输出占用几个单位，内容靠右对齐。如`setw(5) << 10`占用5们位置，10左边有3个空格。
```cpp
#include <iomanip>

cout << setw(5) << 10;
```

## 循环
循环条件定义时，需要指定初始值，如：  
```cpp
int i = 0;
while (i < 10) { ... }
```

## 变量
全局变量定义时，需要一个一个定义。  
```cpp
int a = b = c = 0;   // error!!!

int a, b, c;
a = b = c = 0;      // error!!!

int a = 0;
int b = 0;
int c = 0;         // ok
```
